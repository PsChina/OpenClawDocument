# 第 3 章：快速上手（5分钟体验）

本章将用 7 个 5 分钟内快速上手与 AI 交互的方式。关键要点包括：1. 首次对话、打开 Web 控制台、输入"你好"，请介绍...

## 3.1 首次对话

打开 Web 控制台，输入：

```
文本
你好，请介绍一下你自己
```

AI 会回复它的基本能力和你的配置状态。

## 3.2 尝试一个简单任务

告诉 AI：

```
文本
列出我桌面上的文件
```

AI 会调用 `fs-read` Skill，返回桌面文件列表。如果提示不安装该 Skill，可以手动安装：

```bash
openclaw skills install fs-read
```

## 3.3 让 AI 帮你创建一个文件

在文档目录下创建一个名为"项目资料"的文件夹：

```
文本
在文档目录下创建一个名为"项目资料"的文件夹
```

AI 会调用 `fs-write` Skill 执行操作，并反馈结果。

## 3.4 通过命令行快速提问

你也可以直接接用命令行与 AI 交互（无需打开浏览器）：

```bash
openclaw agent --message "现在几点了？"
```

## 3.5 查看日志

如果遇到问题，查看日志：

```bash
openclaw logs --tail 50
```
